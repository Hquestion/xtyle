Typography

Font

Xtyle uses the excellent Source Sans Pro font from Google Webfonts as its default body font. You can change this option at any time on the xtyle.less file.

Xtyle uses <strong> and <em> tags which should be used when you want to give emphasis to words and sentences. The <b> and <i> tags should be used only for design elements.

Paragraphs

Paragraphs are defined using the <p> tag. Use paragraphs to separate your texts.

First paragraph. This a strong text, this is an italic text, and this is a thin text example.

Another paragraph example. 

<p>First <mark>paragraph</mark>. This a <strong>strong</strong> text, this is an <em>italic text</em>, and this is a </thin>thin text<thin> example.<p>

Drop caps

To capitalize the first word of your paragraph you can use drop caps. You can define a drop caps using the <span class="first-letter"> tag inside a paragraph.

Blockquotes

Write quotes using the <blockquote> tag. Use paragraphs <p> inside blockquote to write content on it. You can also include the source of your quote using the <cite> tag.

"A blockquote."

<blockquote>
  <p>"A blockquote."</p>
</blockquote>

"Right side blockquote."

<blockquote class="right">
  <p>"Right side blockquote."</p>
</blockquote>

"With citation."

– by xchema 
<blockquote>
<p>"With citation."</p>
<cite>
  &#8211;by xchema</cite>
</blockquote>
