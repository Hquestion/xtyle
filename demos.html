Xtyle is a simple HTML, CSS, and JavaScript framework for creating amazing web applications created and maintained by xchema.

Typography

Font

Xtyle uses the excellent Source Sans Pro font from Google Webfonts as its default body font. You can change this option on xtyle.less file.

Xtyle uses <strong> and <em> tags which should be used when you want to give emphasis to words and sentences. The <b> and <i> tags should be used only for designing layouts and interface elements.

Paragraphs

Paragraphs are defined using the <p> tag. Use paragraphs to separate your texts.

First paragraph. This a strong text, this is an italic text, and this is a thin text example.

Another paragraph example. 

<p>First paragraph. This a <strong>strong</strong> text, this is an <em>italic text</em>, and this is a </thin>thin text<thin> example.<p>
<p>Another paragraph example.<p>

Blockquote

Write quotes using the <blockquote> tag. Use paragraphs inside a blockquote to write content on it.

"This is a blockquote."

<blockquote>
  <p>"This is a blockquote."</p>
</blockquote>

"This is a blockquote."

<blockquote class="right">
  <p>"This is a blockquote."</p>
</blockquote>
