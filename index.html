Getting started

Download xtyle and place the files inside of your project folder. This is how a project with xtyle will look like: 

 Your Awesome Project
  … index.html
   xtyle
     css
    … xtyle.css
    … xtyle.min.css
     img
    … favicon.ico
     js
    … xtyle.js
    … xtyle.min.js

Manual installation

You many prefer not to use the index.html provided with xtyle. In this case you can install xtyle manually simple placing the following code on your head tag of your index.html file. 

<link rel="stylesheet" type="text/css" href="xtyle/css/xtyle.min.css">
<script href="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js">
<script href="xtyle/js/xtyle.min.js">
Xtyle uses jQuery to render some HTML elements. You can still put the JavaScript files on the bottom of your document, but that may cause rendering issues.
