HTML,CSS, and JavaScript

If you are a web designer you will feel comfortable using xtyle.

Getting started

Download xtyle and copy the files into your project folder. This is how a project with xtyle will usually look like: 

 Your Awesome Project
  … index.html
   xtyle
     css
    … xtyle.css
    … xtyle.min.css
     img
    … favicon.ico
     js
    … xtyle.js
    … xtyle.min.js

Manual installation

If you decide not to use the index.html provided with xtyle, you can still insert xtyle manually into your project by simple copying the following code to the head HTML tag of your index.html. 

<link rel="stylesheet" type="text/css" href="xtyle/css/xtyle.min.css">
<script href="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js">
<script href="xtyle/js/xtyle.min.js">
