Forms

Structure

All form elements should be used within a <form> tag. 

<form>
  <label>Label</label>
  <input type="text" placecholder="Text input">
  <input type="submit" class="inverse">
</form>

Input fields

Text


<input type="text" placecholder="Text input">
<input type="text" placecholder="Text input rounded" class="rounded">
Sizes



<input type="text" placecholder="Small" class="small"><br>
<input type="text" placecholder="Medium" class="medium"><br>
<input type="text" placecholder="Large" class="large">
Password

<input type="password" placecholder="Password">

Disabled

<input type="text" placecholder="Disabled input" disabled="disabled">

Textarea


<textarea placecholder="Textarea content..."></textarea>
<textarea placecholder="Textarea no resize..." class="noresize"></textarea>
Radio buttons


<input id="radio_male" type="radio" name="gender" value="male" checked="checked">
<label for="radio_male">Male</label>
<input id="radio_female" type="radio" name="gender" value="female">
<label for="radio_female">Female</label>

Checkbox


<input id="check_apples" type="checkbox" name="grocery" value="apples" checked="checked">
<label for="check_apples">Apples</label>
<input id="check_oranges" type="checkbox" name="grocery" value="oranges">
<label for="check_oranges">Oranges</label>

Select*

<select>
<option value="apples" selected="selected">Apples</option>
<option value="oranges">Oranges</option>
</select>
<i class="icon-down-dir select"></i>

*Include <i> after the <select> tag. That is how you differentiate a select box from other buttons.

Fieldset

You can group form elments using the <fieldset> tag element.

Group 

<form>
  <fieldset>
    <legend>Group</legend>
    <input type="text" placeholder="First name">
    <input type="text" placeholder="Last name"><br>
    <input type="submit">
  </fieldset>
</form>

Labels

Labels are the perfect way to give an idea for your users how to fill the input fields.

<label>Label</label>
<input type="text" placecholder="Text input">  



<label class="block">Label</label>
<input type="text" placecholder="Text input">  


<label class="vertical-top">Label</label>
<textarea placecholder="Textarea content..."></textarea>

Buttons

You can create buttons using the <button> tag, using an input type="submit", or creating a link with class .btn.

Link 
<button>button</button>
<input type="submit">
<button class="rounded">Rounded</button>
<a href="#" class="btn">Link</a>
Colors
<button class="inverse rounded">Inverse</button> 
<button class="warning rounded">Warning</button> 
<button class="error rounded">Error</button>
<button class="info rounded">Info</button>
<button class="success rounded">Success</button>
